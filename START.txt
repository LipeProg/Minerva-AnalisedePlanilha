DataDash - Inicio Rapido

1. Leia `COMECE_AQUI.md`
2. Fluxo minimo:

   cd /caminho/para/Minerva-AnalisedePlanilha
   python3 -m venv venv
   source venv/bin/activate
   python3 -m pip install -r requirements.txt
   python3 datadash/manage.py migrate
   python3 datadash/manage.py runserver

3. Abra `http://127.0.0.1:8000`
4. Teste com `exemplo_vendas.csv`

Observacao:
- O projeto usa `sqlite3` por padrao
- MySQL ficou opcional via variaveis de ambiente
