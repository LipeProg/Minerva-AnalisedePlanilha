## 🖨️ DICAS DE IMPRESSÃO

### Para impressora comum
1. Use tamanho A4
2. Configure margem padrão (2,5 cm)
3. Impressão dupla-face economiza papel
4. Use fonte preta em fundo branco

### Para melhor apresentação
1. Imprima colorido se possível
2. Use capa dura para primeira página
3. Coloque espiral ou clipe

### Se quiser economizar
1. Imprima apenas CAPA + RESUMO (primeiras 5 páginas)
2. DOCUMENTACAO_IMPRESSAO.md tem 25 páginas, imprima apenas seções importantes
